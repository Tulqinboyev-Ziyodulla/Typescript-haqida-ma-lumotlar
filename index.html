TypeScript haqida ma'lumotlar

TypeScript - bu JavaScriptning kuchaytirilgan versiyasi bo'lib, dastur kodining sifati va xavfsizligini oshiradi.

Asosiy afzalliklari

  • Statik tiplash: Ma'lumotlarning turini aniq belgilash va xatoliklarni oldindan aniqlash imkonini beradi.
  • Modern funksiyalar: Class, Interface, Enum va Decorators kabi imkoniyatlarni qo'llab-quvvatlaydi.
  • Keng platforma qo'llab-quvvatlovi: Node.js, React, Angular kabi frameworklarda ishlatiladi.
  • JavaScript bilan to'liq moslik: JavaScript kodini TypeScript ichida ishlatish mumkin.

O'rnatish va ishga tushirish

TypeScript ni o'rnatish:

npm install -g typescript

tsconfig.json faylini yaratish:

npx tsc --init

Watch mode ni ishga tushirish:

npx tsc -w
